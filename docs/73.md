# 2025年10月18日-2025年10月25日


## 1.ExplorerPatcher

虽然 Win11 目前已经成为 用户最多的电脑操作系统，

但是很多人并不是很喜欢它的设计，尤其是任务栏和右键菜单。

ExplorerPatcher 是一个开源的系统优化工具，

可以让我们定制 Win11 的很多设计。


打开它之后，电脑会自动重启 Explorer 窗口，此时屏幕可能会黑屏几秒，

然后你的任务栏就会变成 Win10 的样式。


在下边的任务栏中点击右键，选择 “属性”，打开 ExplorerPatcher 的设置页面，

能在这里看到很多其它的选项，这些你都可以按自己的想法调整。


比如能选择使用 Win10 还是 Win11 的风格，是否显示任务栏上的搜索框，

禁用掉 Win11 的双层右键菜单，恢复 Win10 的磁贴，

修改窗口切换器样式等几十种可调节选项。


ExplorerPatcher（Win11优化工具）
https://github.com/valinet/ExplorerPatcher


## 2.ossnav


这是一个开源项目精选合集，

我看了一下里边的内容，确实都是比较优质的项目。

比如知名的 Steam 工具箱 Watt Toolkit，

免费开源的录屏、直播软件 OBS Studio，

微软官方开源的 Windows 工具箱 PowerToys，

开源的远程控制软件 RustDesk，

如果你对各种有用的开源项目感兴趣，可以了解一下其中的内容。


ossnav（开源精选）
https://github.com/maxiaobang7/ossnav

## 3.wechat-selkies


有人把 Linux 版本的微信使用 Docker 进行了封装，

并且用 Selkies WebRTC 提供了浏览器支持。

当你把它部署到服务器上之后，你就得到了一个可以随时网页访问，并且实时在线的网页版微信。

因为 Linux 版微信使用的是和 Windows 一样的框架，

所以功能上也没有太多缺失的地方。


此前的网页版微信目前已经基本上不能用了，

这个稀奇古怪的办法，可能是唯一能在浏览器上用微信的渠道了。


wechat-selkies（网页版微信）
https://github.com/nickrunning/wechat-selkies

## 4.preserve-cd


这个项目的中文名叫 “绝版游戏保护工程”，

看名字就知道，里边保存的都是一些比较珍惜的老游戏资源，

并且，作者并不是把各种内容都一股脑的塞进去，而是进行了精选，

比如像 雷电、红警、仙剑奇侠传、金庸群侠传、三国志等都是经典游戏，点开就能玩。

而且作者还打包了模拟器和游戏攻略，非常简单纯粹。


preserve-cd（绝版游戏保护工程）
https://github.com/skywind3000/preserve-cd


## 5.AI-Video-Transcriber


这是一个开源的 AI 视频转录器，

可以一键提取 30 多个视频平台的视频文案，并且使用起来非常简单，

把它运行起来，然后复制一下视频链接，粘贴到输入框中，

点击 “Start”，稍微等一会儿，就可以获取到视频的字幕文案，

如果填写一下 AI 的密钥，还可以获取到 AI 的总结纪要。


AI-Video-Transcriber（视频文案转录）
https://github.com/wendy7756/AI-Video-Transcriber
